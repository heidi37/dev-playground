Day 5

JavaScript Modules

The index.html file is importing a new .js file as a type="module". This file gets the pages array from another .js file.


        

        
          <script src="/js/index.js" type="module"></script>
         
        
This is the code in index.js to import the array

        
          import { pageLinks } from './js/page-array.js';
        
        
This is the code in page-array.js that exports the page array

        
          export { pageLinks };
        
        
The meta refresh is getting generated dynamically

        
          const meta = document.createElement("meta");

          document.head.appendChild(meta);
          
          meta.setAttribute("http-equiv", "Refresh");

          meta.setAttribute
          ("content", `0; url='days/day-${pageLinks[index]}/
          day-${pageLinks[index]}.html'`)
