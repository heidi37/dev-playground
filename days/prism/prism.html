Prism

I downloaded prism so I could show source code on my web pages in an intelligible way..


  p { 
      color: red;
      padding: 25px;
    }

To use prism you need to download the .js and .css files and include them in your documents. You have choices as to which languages and plugins you want included in these files. The less you choose the smaller the files. Here is how you include the downloaded files


  <!DOCTYPE html>
  <html>
  <head>
    ...
    <link href="themes/prism.css" 
    rel="stylesheet" />
  </head>
  <body>
    ...
    <script src="prism.js"></script>
  </body>
  </html>

This is how flag your text in the html to use prism


  <pre><code class="language-css">
  p { color: red }</code></pre>

Caveat: it is a little more complicated when you want to show html code. Because you are in an html document it doesn't want to display on the page the "<" & ">". So you have to find and replace them in the text you want to display on the page.
